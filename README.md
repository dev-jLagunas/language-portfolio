# JLAG English Coach

This is a professional landing page for my English coaching practice. It is designed to convert students by presenting a clear, structured methodology through a clean and functional interface.

## Project Overview

As a Product Designer, I built this site to serve as a high-performance digital home for my coaching business. The focus is on straightforward information architecture and a user experience that feels reliable and easy to navigate across different cultures.

### Key Features

- **Full Localization:** The site is fully translated into **English, Spanish, Japanese, and French**. The visual theme and color palette adapt dynamically to match the selected language.
- **Unified Navigation:** I built a synchronized navigation system that keeps the mobile menu, the desktop scroll indicator, and the sidebar "elevator" in sync using a shared global state.
- **Brutalist Design System:** A custom CSS framework using a high-contrast, bold aesthetic. It relies on a robust set of variables to keep the design consistent and professional.
- **Interactive Methodology:** I used **GSAP** to create a horizontal scroll section for my coaching framework, making the process easy to follow and visually distinct.
- **Custom Visuals:** The site features custom polaroid-style testimonials and specific icon mapping to create a tactile, personalized feel.

## Performance & Build

- **Speed & Optimization:** All images are in **.webp** format for fast loading. I used **SVG filters** for background grain textures to keep the site light and high-performance.
- **Responsive Design:** The site is refactored for every screen size, ensuring the layout and animations work perfectly on both mobile devices and large monitors.
- **Clean Tech Stack:** Built with Nuxt 4 and Vue 3, utilizing modern CSS standards for long-term maintainability.

## Tech Stack

- **Nuxt 4 / Vue 3**
- **GSAP** (ScrollTrigger)
- **Nuxt i18n**
- **Nuxt Icon**
- **Google Fonts**
